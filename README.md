### 设计思路

rpanel 通过 subscribe redis 的一个 "RPANEL" channel, 数据源往该通道中写入数据，
rpanel接受后实时在画板中画图。

数据需要跨外网传递时，建议使用ssh, ngrok之类的反向代理。

### 当前支持的指令

* ADD duration x y

  往画板中添加点 x y，持续时间 duration 秒，为 0 时一直添加

### 当前支持的快捷键

* R 清空画板
